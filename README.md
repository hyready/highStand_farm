# 介绍
### 目录结构
按照umi的约定，目录结构如下
```
.
├── dist/                          // 构建产物目录
├── config/                        // webpack配置等
├── public/                        // index.html等构建文件
└── src/                           // 源码目录，可选，把里面的内容直接移到外面即可
    ├── css/                       // 公共样式
    ├── components/                // 组件
```
